Teacher Assistant ‒ Certification Notes for
Microsoft Reviewers

No account or login is required to use the add-in.

How to Install and Open the Add-in
	1. Open Microsoft PowerPoint (Windows Desktop, version 16.0 or later)
	2. Go to the Home tab in the ribbon
	3. Click the "Teacher Assistant" button
	4. The taskpane sidebar opens on the right side of the screen

How to Test the Core Flow
	Step 1 — Configure settings
		On first launch, a settings modal opens automatically
		Set Language to English, Level to B1, leave Age Group empty
		Click Save
		The modal closes and the context badge in the header shows "B1 English"

	Step 2 — Generate slides
		Type in the input field: 
			Generate explanation of Present Simple 3 slides include examples
		Press Enter or click the send button
		Wait for the AI to respond (3‒30 seconds depending on server state)
		A slide preview appears showing a title slide and 3 content slides

	Step 3 — Navigate the preview
		Use the Next and Back buttons to navigate between slides
		On the last slide the Next button changes to "Insert 6"

	Step 4 — Insert slides into PowerPoint
		Click "Insert 3" on the last slide
		Slides are inserted into the active presentation
		A success message confirms insertion

Additional Features to Verify

	Edit a slide
		After generating slides, navigate to any slide and click Edit
		Type an instruction such as: 
			Make the explanation simpler
		Press Enter — the slide updates in place

	Cancel generation
		While the progress indicator is showing, click Cancel or press Q
		Generation stops and the input becomes active again

	New chat
		Click the New Chat button at the top of the taskpane to reset the conversation
